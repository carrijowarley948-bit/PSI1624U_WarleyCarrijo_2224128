# 🎮 Arcade

## Sobre o Projeto

Arcade é uma aplicação desenvolvida em C# utilizando Windows Forms que reúne vários jogos numa única plataforma.

O objetivo do projeto é proporcionar uma experiência simples e organizada, permitindo ao utilizador criar uma conta, iniciar sessão, jogar diferentes minijogos e consultar as suas pontuações.

## Funcionalidades

- Registo e Login de utilizadores;
- Sistema de contas;
- Biblioteca de jogos;
- Placar de pontuações;
- Reprodução de música.

## Jogos Disponíveis

- Damas
- Pong
- Zumbis

## Tecnologias Utilizadas

- C#
- Windows Forms
- .NET Framework 4.7.2
- Visual Studio

## Como Executar

1. Abrir a solução no Visual Studio.
2. Restaurar os pacotes necessários.
3. Compilar o projeto.
4. Executar a aplicação.
